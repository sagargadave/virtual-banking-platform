# Virtual Banking Platform

A Full-Stack Virtual Banking Application that provides secure account management, fund transfers, transaction tracking, profile management, and PDF account statements.

## Features

### Authentication & Security

* User Registration
* User Login
* JWT Authentication
* Route Protection
* Password Encryption using BCrypt
* Spring Security Integration

### Banking Operations

* Deposit Money
* Withdraw Money
* Transfer Funds Between Accounts
* View Account Details
* Account Balance Tracking

### User Management

* View Profile
* Update Profile
* Change Password

### Transaction Management

* Transaction History
* Recent Transactions Dashboard
* Transaction Type Tracking

### Reports

* Generate PDF Account Statement

## 🔐 Authentication Flow

1. User registers an account.
2. User logs in with email and password.
3. Backend generates JWT Token.
4. Token is stored in Local Storage.
5. Protected routes are accessed using JWT Authentication.
6. Spring Security validates every request.
